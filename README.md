# buzonQuejas
Sistema de gestión de quejas desarrollado con Vue, PHP y MySQL

Título: Sistema Búho de Gestión de Quejas

Aplicación web desarrollada como proyecto escolar para el Instituto Tecnológico Nacional de México, campus Orizaba.  
El sistema permite a los estudiantes registrar quejas o sugerencias de manera anónima y darles seguimiento, mientras que los administradores pueden gestionar y actualizar el estado de cada queja.

El proyecto fue desarrollado bajo una arquitectura MVC, utilizando Vue.js para el frontend, PHP para el backend y MySQL como sistema gestor de base de datos.

# Tecnologías utilizadas
- Vue.js (JavaScript)
- Vue Router
- API y Composition API
- PHP
- MySQL
- XAMPP (Apache y MySQL) 
- Vite

# Gestión de administradores
Por motivos de seguridad, el registro de usuarios con rol de administrador no se realiza desde la interfaz principal del sistema.
Los administradores se insertan directamente en la base de datos mediante scripts controlados del backend, utilizados únicamente durante la configuración inicial del sistema en un entorno local.
